Using Sorts and Custom Sorts in Excel

The sorting tool provided in MS Excel allows you to rearrange data of various types into any given order. When given columns with text data this tool sorts them in ascending or descending alphabetical order. Similarly, when given numerical data, it sorts the column from smallest to largest or Largest to smallest. Furthermore, custom sorts allow you to perform more advanced sorting by providing more sorting criteria. Some functions of custom sorts are performing the sort based on a custom list or by format, such as cell color.

The sorting tool in Excel is integral to using the application to its fullest potential. The tool provides ways to efficiently arrange data and helps in scenarios where you need to quickly analyze data, identify trends, or present your data in a more structured manner. For example, you may want to order a list of dates in order from oldest to newest or a list of names in alphabetical order. By using the sorting tool, you can quickly organize your data to be better visualized and understood.

Sorts

Step 1 - Choose Column

Click on a cell in the column that you want to sort. Here, I am using the apple quality sheet and want to sort using the Size column.

Step 2 - Home -> Sort & Filter

Go to the Home tab in Excel’s ribbon at the top of the screen. Click the Sort & Filter button on the right side of the ribbon.

Step 3 - Choose Sort

Now you are given a few sort options depending on the type of data in the chosen column. Here, these options are sorting numbers from “Smallest to Largest” or “Largest to Smallest.” These options sort numerical values in ascending and descending order. If the selected column has numerical data, the options to be sorted would be sorting “A to Z” or “Z to A” These options sort the data alphabetically in either ascending or descending order. If the selected column contains dates and times, the options to be sorted would be sorting “Oldest to Newest” or “Newest to Oldest.” These options sort date and time values in ascending and descending order. I chose to sort my apple sizes from largest to smallest.

This is the sheet sorted on the Size column from largest to smallest.


Custom Sorts

Step 1 - Home -> Sort & Filter -> Custom Sort

Click anywhere inside your Excel sheet’s data. Then go to the Home tab in Excel’s ribbon at the top of the screen. Click the Sort & Filter button on the right side of the ribbon and then choose Custom Sort from the drop down menu. I am using the Grocery Store sheet for this custom sort.

Step 2 - Column -> ‘Column Name'

On the Sort window that popped up, you can add multiple levels of sorts. Click on the drop down menu under Column, directly right of ‘Sort by’ and then pick the column you want to use to sort by. I selected the Sub Category column to sort.

Step 3 - Sort On -> Cell Values

In the drop down menu under Sort On, select what you want to use in the sorting. Here we are sorting by cell values, but you can also sort on Cell Color, Font Color, and Conditional Formatting Icon. I selected the option to sort on the Cell Values.

Step 4 - Order

In the drop down menu under Order, select what order to sort the data into. For text data, the drop down options are “A to Z”, “Z to A,” and “Custom List…” For Numerical data, the drop down options are “Smallest to Largest”, “Largest to Smallest,” and “Custom List…” I selected the option to sort from A to Z so the subcategories will be sorted in ascending alphabetical order.

Step 4.a - Order -> Custom List... When sorting by Custom List, click on the Custom List… option.

Step 4.b - NEW LIST -> Add -> OK Here, you can choose from a variety of predefined lists, or create your own custom list. To create a new custom list, click on NEW LIST and then type out the list in the empty white box titled “List entries:” I selected the option to sort from A to Z so the subcategories will be sorted in ascending alphabetical order. To complete the list, click Add. Finally, choose which list you want to use, and then click OK to add it to the dropdown menu for Order.

Step 4.c - Select Order Now you can use it to sort by selecting it in the Order dropdown menu.

Step 5 - Add Level

Select Add Level at the top left corner of the window to create a new row of sorts. This will add another level of organization in the custom sort. Follow steps 2-4 again to complete the new sort.

Note: You can add as many levels as needed to the custom sort, but the sequence of rows matters and you can move them as needed to perform the sorts in the correct order.

Step 6 - Complete Custom Sort

Repeat Step 5 until you have all levels of sorting completed and then select OK for Excel to execute the sort on the sheet.

This is the sheet sorted according to the custom sort we created above. Here, we can see that the Sub Category column was sorted alphabetically first, and then the prices column was sorted in ascending order.
